## 🔥 Live Projects

### 🌐 Novex — AI Model Comparison Desktop App

> One prompt, four AI models at once. A cross-platform Electron app that keeps ChatGPT, Claude, Gemini & Perplexity loaded simultaneously — dispatching prompts in parallel and delivering responses in under 200ms with zero-reload tab switching.

[![Novex Live Demo](https://api.microlink.io/?url=https://novexhub.netlify.app&screenshot=true&meta=false&embed=screenshot.url)](https://novexhub.netlify.app/)

- ⚡ Zero-API prompt injection via Electron contextBridge IPC with a 4-level DOM selector fallback chain (data-testid → aria-label → CSS class → XPath), resilient to site layout changes
- 🔒 Privacy-first local session model — no backend calls, no credential storage, prompt history recall via keyboard navigation, global hotkey summon in under 30ms

`Electron` · `Node.js` · `JavaScript` · `Netlify`

### 🗺️ GeoForge — 2D Map to 3D City Model Pipeline

> Type any city name, get a rendered 3D model — zero manual data preparation.

[![GeoForge Live Demo](https://api.microlink.io/?url=https://huggingface.co/spaces/Moe32/Geoforge&screenshot=true&meta=false&embed=screenshot.url)](https://huggingface.co/spaces/Moe32/Geoforge)

- 🛰️ Chains 5 free geospatial sources — Nominatim (geocoding), Overpass API (building footprints), Open-Elevation & NASA SRTM (terrain) — into a full resolution pipeline
- ⚡ SciPy KDTree spatial index samples LiDAR elevation stats per polygon across millions of 3D points, resolving all 8 geometry types in under 2.15s per city model
- 📦 Public Flask REST API with 9 endpoints and 6 export formats (OBJ, CityJSON 2.0, CityGML 2.0, STL, PostGIS SQL, CSV), containerized with Docker and deployed on Hugging Face Spaces with real daily users

`Python` · `Flask` · `Three.js` · `GeoPandas` · `Docker` · `Hugging Face Spaces`

### 💬 Java Real-Time Chat Application

> A multithreaded TCP chat with live typing indicators and voice-to-text — built with zero external dependencies beyond JDK 17.

![Java Chat Demo](assets/java-chat-demo.png)

- 🧵 Multithreaded TCP relay server with a dedicated thread per client, broadcasting a 5-event plain-text protocol (JOIN, MSG, TYPING, STOP_TYPING, LEFT) with zero missed messages
- ⌨️ Dual-signal protocol with live typing indicator (1.2s cooldown per keystroke, capping traffic at 1 event per typing burst) and a message-integrity badge
- 🎙️ Zero-library voice input pipeline: local WAV recording via `javax.sound.sampled`, transcription through OpenAI Whisper over raw `HttpURLConnection`

`Java` · `Sockets` · `Multithreading` · `Swing` · `OpenAI Whisper`
